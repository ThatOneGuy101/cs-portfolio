How To Page 2

Today, I will teach you how to make an interactive mouse program. To make this, make sure you have a draw function, if and else conditions, fills, strokes and strokeWeights

When you are finished with the coding, it should look like this:


    draw = function() {
    background();
    
    if (mouseX < 76) {
        ellipse(mouseX, mouseY, 30, 50);
    }
    else if (mouseX > 151) {
        ellipse(mouseX, mouseY, 30, 30);
    }
    else {
        rect(mouseX, mouseY, 30, 30);
    }
    // will determine the color of the circle depending on where your mouse is
    if (mouseX < 76) {
        fill(0, 210, 247);
        strokeWeight(3);
        stroke(57, 0, 214);
    }
    else if (mouseX < 151) {
        fill(0, 21, 27);
        strokeWeight(1);
        stroke(57, 0, 214);
    }
    else {
        fill(100, 110, 47);
        strokeWeight(2);
        stroke(57, 20, 14);
    }
};

First, the if statements

The if statements will go in the order from up to down. It will go through all of them until it finds a solution through one of the if statements. An example would be if x = 4 then output A, or else output B. That would be an example of an if/else statement.

Second, the fill function

The fill function will color in anything under it. This can be used for making certain shapes a certain color to create such s masterpeice.

Third is the strokeWeight

The strokeWeight increases the amount of width around the shape. You could call it the border or outline.

Last is the stroke

The stroke would be the color of the strokeWeight

After all of this coding, I am sure to remember about the if statements because it's the most important code within the program and because it's fun yo use. The If condition is usually used for making random solutions like choosing between option A or option B.
